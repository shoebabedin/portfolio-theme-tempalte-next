requirements

There are system requirements in order to install and setup the "DesignX Portfolio" and its components properly. Make sure that you are running the node.js install npm install. You can download Node.js from here: Node JS. NPM comes bundled with Node.js

Recommended node and npm

node install

npm install

Install Npm Packages

NPM : You must have to npm install packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.

npm install

Development server

Run npm run start for a dev server. Navigate to http://localhost:3000/ . The app will automatically reload if you change any of the source files

npm run dev

Build

Run npm run build to build the project. The build artifacts will be stored in the build/ directory. Use the npm run build --prod flag for a production build.

npm run build

npm run build --prod

More Help

To get more help on the react use npm --help or go check out the React Support.

npm --help

Change Site Title and Favicon

Go to /public folder and replace your icon file with the file named favicon.ico to change the fav icon.

Go to /pages/index.js component and change title.

Change Logo

Go to /components/Header/Header.js file and change logo.

Customize Menu

Go to /components/Header/Header.js file and change menu items.

Customize Footer

Go to /components/Footer/Footer.js file and change footer logo, social media, footer data.

Change Copyright Infomartaion

Go to /components/Footer/Footer.js file and change copyright information.

Section Components

Go to /components folder. In components folder you will get all the section components. Go through each components and change section data.

Social Networks

Go to /components/Footer/Footer.js file to change social links.

Typography - Font Family

Go to /styles/global.css file and put your font family link to change the font of the website.
